#### +号的使用
1. 加号左右两边都是数值时，做加法运算
2. 加号左右两边有一方是字符串时，做拼接运算
``` 
import java.lang.reflect.Parameter;

public class demo3_var3 {
    public static void main(String[] args) {

        int a = 50;
        int b = 20;
        String s = "hello";

        System.out.println(a + b);
        System.out.println(a + s);

    }
}

结果
70
50hello
```
