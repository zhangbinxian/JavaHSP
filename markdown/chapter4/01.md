##### 嵌套分支结构
嵌套分支结构,在一个分支结构当中嵌套了另一个完整的分支结构.<br>
规范:不要超过3层,可读性不好
```
if(){
    if(){
        //if else
    }else{
        //if else
    }
}else{
    //if else
}
```
switch-case
``` 
switch(表达式){
    case 常量1:   //当...,执行语句
        语句块1:
        break;    //退出switch
    case 常量2:
        语句块2: 
        break;  
    case 常量3:
        语句块3:
        break;
        
        ......    
    case 常量n:
        语句块n: 
        break;
    default:
        default语句块;
        break;  
  
}
```
default和break的三种特殊情况
